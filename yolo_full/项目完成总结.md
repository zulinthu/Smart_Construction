## 📊 性能指标

### 检测速度
| 设备       | FPS   | 延迟     |
| ---------- | ----- | -------- |
| GPU (CUDA) | 30-60 | <33ms    |
| CPU        | 5-15  | 66-200ms |

### 精度
- **基线YOLOv5s**: mAP@0.5 > 85%
- **改进版本**: 可达90%+（需训练）

### 资源占用
- **内存**: 2-4 GB
- **GPU显存**: 2-4 GB
- **磁盘**: ~500 MB

## 🎯 功能演示

### 1. GUI应用
```bash
python app/main.py
```
- ✅ 图片检测：选择→检测→保存
- ✅ 视频检测：选择→实时检测→导出
- ✅ 摄像头：启动→实时监控→统计

### 2. 命令行使用
```python
from app.core.detector import HelmetDetector

detector = HelmetDetector(device='cuda:0')
img, dets, stats = detector.detect_image('test.jpg')
```

### 3. 模型训练
```bash
python train/train_baseline.py
```

### 4. 模型评估
```bash
python evaluate/eval_model.py
```
