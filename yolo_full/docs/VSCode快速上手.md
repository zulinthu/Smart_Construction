# VSCode快速上手（5分钟教程）

## 🚀 30秒快速启动

### 方法1：使用快捷键（最快）⚡
```
1. 打开VSCode
2. 按 Ctrl+Shift+R
3. 应用启动！
```

### 方法2：使用状态栏按钮
```
1. 打开VSCode
2. 点击底部状态栏任务图标
3. 选择 "▶️ 启动GUI应用"
```

### 方法3：使用命令面板
```
1. 按 Ctrl+Shift+P
2. 输入 "Tasks: Run Task"
3. 选择 "▶️ 启动GUI应用"
```

## 🐛 调试教程

### 1. 设置断点
```
在代码行号左侧点击，出现红点 ●
```

### 2. 开始调试
```
按 F5 或点击左侧调试图标
选择调试配置（如"启动GUI应用"）
```

### 3. 调试控制
```
F10 - 单步跳过（不进入函数）
F11 - 单步进入（进入函数）
Shift+F11 - 跳出函数
F5 - 继续运行
Shift+F5 - 停止调试
```

### 4. 查看变量
```
调试时，鼠标悬停在变量上可以查看值
或在左侧调试面板的"变量"区域查看所有变量
```

## 💡 实用技巧

### 技巧1：多文件快速切换
```
Ctrl+P 输入文件名，快速打开
Ctrl+Tab 在最近的文件间切换
```

### 技巧2：查找定义
```
F12 - 跳转到定义
Alt+F12 - 查看定义（不跳转）
Shift+F12 - 查找所有引用
```

### 技巧3：多行编辑
```
Alt+点击 - 添加多个光标
Ctrl+Alt+↑/↓ - 上下添加光标
Ctrl+D - 选择下一个相同的词
```

### 技巧4：快速注释
```
Ctrl+/ - 注释/取消注释单行
Shift+Alt+A - 注释/取消注释多行
```

### 技巧5：代码折叠
```
Ctrl+Shift+[ - 折叠当前区域
Ctrl+Shift+] - 展开当前区域
Ctrl+K Ctrl+0 - 折叠所有
Ctrl+K Ctrl+J - 展开所有
```

## 🔧 首次配置清单

### 1. 安装推荐扩展（2分钟）
```

```


## 🚨 常见问题速查

### Q: 快捷键不工作？
**A:** 检查是否与其他软件冲突，关闭其他可能占用快捷键的软件

### Q: 看不到任务按钮？
**A:** 按 Ctrl+Shift+P → "Tasks: Run Task" 可以手动选择

### Q: Python找不到？
**A:** 按 Ctrl+Shift+P → "Python: Select Interpreter" 重新选择

### Q: 终端中文乱码？
**A:** 设置 → 搜索"terminal encoding" → 选择 utf8

### Q: 格式化不工作？
**A:** 确保已安装Black：`pip install black`
