## 🔧 常见问题解决

### Q1：GPU不可用
```bash
# 检查CUDA
python -c "import torch; print(torch.cuda.is_available())"

# 如果返回False，检查CUDA安装
nvidia-smi
```

**解决方案**：
- 安装对应版本的CUDA Toolkit
- 重新安装PyTorch（GPU版本）
- 或使用CPU训练（较慢）

### Q2：内存不足
**解决方案**：
- 减小batch size：`--batch-size 8`（或更小）
- 减小图像尺寸：`--img-size 416`
- 使用渐进式训练

### Q3：摄像头打不开
```bash
# 测试摄像头
python -c "import cv2; cap = cv2.VideoCapture(0); print('成功' if cap.isOpened() else '失败')"
```

**解决方案**：
- 检查摄像头是否被其他程序占用
- 尝试不同的摄像头ID（0, 1, 2...）
- 重启电脑
- 检查摄像头权限

### Q4：训练很慢
**解决方案**：
- 使用GPU训练
- 减小batch size但保持总样本数
- 使用更小的模型
- 减少训练轮数先测试

### Q5：PyQt5安装失败
```bash
# Windows
pip install PyQt5 -i https://pypi.tuna.tsinghua.edu.cn/simple

# 或使用conda
conda install pyqt
```
