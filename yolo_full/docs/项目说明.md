#### 5.2 摄像头检测实现（核心）
```python
# 在 app/core/detector.py 中完善
def detect_camera(self, camera_id=0):
    """实时摄像头检测"""
    cap = cv2.VideoCapture(camera_id)
    
    while True:
        ret, frame = cap.read()
        if not ret:
            break
        
        # 检测
        detections = self.detect(frame)
        
        # 绘制结果
        result_frame = self.draw_detections(frame, detections)
        
        # 统计分析
        stats = self.calculate_statistics(detections)
        
        # 返回结果
        yield result_frame, detections, stats
```

#### 5.3 连接界面和检测器

### 可能遇到的问题与解决方案

#### 问题1：数据集不足
**解决方案**：
- 使用数据增强扩充数据集
- 结合多个公开数据集
- 使用迁移学习

#### 问题2：精度不达标
**解决方案**：
- 调整超参数
- 增加训练轮数
- 使用更大的模型
- 改进数据标注质量

#### 问题3：实时性不够
**解决方案**：
- 使用更轻量的模型
- 降低输入分辨率

- 使用TensorRT加速

#### 问题4：摄像头兼容性
**解决方案**：
- 测试多种摄像头
- 提供摄像头选择功能
- 提供参数调节选项
- 降级OpenCV版本

#### 问题5：GPU内存不足
**解决方案**：
- 减小batch size
- 使用梯度累积
- 使用混合精度训练
- 使用模型并行
