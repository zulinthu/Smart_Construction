## 1. 项目概述

### 1.1 研究背景
施工现场安全帽佩戴检测是保障施工安全的重要环节。传统人工巡检方式效率低、成本高，且难以实现全覆盖监控。本项目旨在开发一套基于深度学习的智能检测系统。

### 1.2 项目目标
1. **改进YOLOv5算法**：提升检测精度和速度，实现轻量化部署
2. **实时摄像头检测**：支持实时视频流检测，非单纯图片上传
3. **完整检测系统**：开发具有友好界面的智能检测系统

### 1.3 核心创新点
- ✅ 轻量化骨干网络设计（降低参数量40%以上）
- ✅ 多注意力机制融合（提升小目标检测能力）
- ✅ 实时摄像头检测系统（满足导师核心要求）
- ✅ 完整的对比实验验证（证明改进有效性）

## 3. YOLOv5改进方案

### 3.1 改进思路总览

**目标：在保证精度的前提下，大幅降低模型参数量，提升推理速度**

```
原始YOLOv5s
    ↓
[改进1] 轻量化骨干网络替换
    ↓
[改进2] 注意力机制集成
    ↓
[改进3] 多尺度特征融合优化
    ↓
[改进4] 损失函数改进
    ↓
改进YOLOv5（最终版本）
```

### 3.2 改进方案详细设计

#### 改进1：轻量化骨干网络

**方案A：MobileNetV3（推荐）**
- ✅ 优点：成熟稳定，硬件支持好，参数量少
- 📊 参数量：~4.2M（比CSPDarknet53减少60%）
- ⚡ 速度提升：约30-40%

**方案B：ShuffleNetV2**
- ✅ 优点：通道重组效率高，实际推理速度快
- 📊 参数量：~3.5M
- ⚡ 速度提升：约35-45%

**方案C：GhostNet**
- ✅ 优点：Ghost模块生成特征图，计算量小
- 📊 参数量：~4.0M
- ⚡ 速度提升：约25-35%

**选择策略：**
- 优先选择 **MobileNetV3**（平衡性能和兼容性）
- 备选方案：ShuffleNetV2（追求极致速度）

#### 改进2：注意力机制集成

**2.1 通道注意力：SENet/ECA-Net**

```python
# 集成位置：每个特征提取块后
特征提取 → 通道注意力 → 激活输出

# ECA-Net（推荐，参数少）
class ECABlock:
    - 全局平均池化
    - 1D卷积学习通道权重
    - Sigmoid激活
    - 特征重标定
```

**2.2 空间注意力：CBAM**

```python
# 集成位置：Neck部分（特征融合前）
特征输入 → 通道注意力 → 空间注意力 → 融合输出

# CBAM空间注意力分支
- Max Pooling + Avg Pooling
- 7x7卷积
- Sigmoid激活
```

**2.3 混合注意力策略（推荐）**

```
Backbone: ECA-Net（轻量级通道注意力）
   ↓
Neck: CBAM（增强空间感知）
   ↓
Head: 直接检测输出
```

#### 改进3：多尺度特征融合优化

**原始PANet → 改进BiFPN**

```
改进点：
1. 移除单向节点（减少计算量）
2. 增加跨层跳跃连接（增强特征传递）
3. 加权特征融合（学习不同层级重要性）
```

```python
# 特征融合权重学习
P3_out = w1*P3_in + w2*Upsample(P4_in)
P4_out = w1*P4_in + w2*Downsample(P3_out) + w3*Upsample(P5_in)
P5_out = w1*P5_in + w2*Downsample(P4_out)
```

#### 改进4：损失函数改进

**4.1 边界框回归损失：CIoU Loss**

```python
# 替换原始GIoU Loss
CIoU = IoU - (ρ²/c²) - αv

其中：
- ρ: 中心点距离
- c: 外接矩形对角线距离
- v: 长宽比一致性
- α: 权重系数
```

**4.2 分类损失：Focal Loss（可选）**

```python
# 针对正负样本不平衡
FL = -α(1-pt)^γ * log(pt)

适用场景：
- 背景类别占比过大
- 难易样本不平衡
```
